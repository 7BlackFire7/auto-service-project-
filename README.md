AutoServis

Webová aplikácia pre autoservis vytvorená pomocou frameworku Vue.js.
Projekt slúži na prezentáciu služieb autoservisu a jednoduchú správu objednávok.

Demo stránky:
https://7blackfire7.github.io/auto-service-project-/#/

Popis projektu

Aplikácia AutoServis je jednoduchá SPA (Single Page Application), ktorá umožňuje používateľovi:

prezerať ponúkané služby autoservisu,

vybrať konkrétnu službu,

vytvoriť objednávku zadaním mena a modelu auta,

zobraziť zoznam vytvorených objednávok.

Projekt je určený ako študijný projekt na precvičenie práce s komponentmi, routovaním a základnou logikou aplikácie vo Vue.js.

Použité technológie

Vue.js

Vue Router

HTML5

CSS3

JavaScript (ES6)

Funkcionalita aplikácie

Navigačný panel s routovaním medzi stránkami

Zobrazenie služieb autoservisu s cenami

Formulár na vytvorenie objednávky

Dynamické zobrazovanie objednávok

Responzívny dizajn

Cieľom projektu je:

precvičiť prácu s Vue.js,

naučiť sa používať Vue Router,

vytvoriť jednoduchý, prehľadný a funkčný webový projekt.
